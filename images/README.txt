IMAGES FOLDER
=============

Place your images for the 4 article pages in this folder.

Expected filenames:
  - photovoltaic.jpg         → used in photovoltaic.html
  - origin-destination.jpg   → used in origin-destination.html
  - tourism.jpg              → used in tourism.html
  - vehicle-tracking.jpg     → used in vehicle-tracking.html

Recommended image size: 1600 x 700 px (wide landscape format, 16:7 ratio)
Format: JPG or PNG both work fine.

If an image file is missing, the page will show a placeholder box automatically.
